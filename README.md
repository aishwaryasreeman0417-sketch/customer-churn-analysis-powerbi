# 📊 Customer Churn Analysis Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Customer Churn Analysis Dashboard built using Microsoft Power BI. The dashboard helps businesses understand customer churn behavior by analyzing customer demographics, contract types, internet services, payment methods, tenure, and revenue.

The goal of this project is to identify the key factors contributing to customer churn and provide actionable business insights through data visualization.

---

## 🎯 Objectives

- Analyze customer churn patterns.
- Monitor overall churn rate.
- Identify high-risk customer segments.
- Compare churn across contract types and payment methods.
- Analyze customer demographics.
- Understand the impact of customer tenure on churn.
- Visualize revenue and monthly charges.

## 📈 Key Insights

- Customers with month-to-month contracts have the highest churn rate.
- Fiber optic users are more likely to churn.
- Customers with shorter tenure tend to leave more frequently.
- Electronic check users show comparatively higher churn behavior.
- Long-term contracts improve customer retention.
